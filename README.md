# 校医院管理系统

## 项目简介
校医院管理系统采用SSM（SpringMVC Spring Mybatis）框架开发,数据库采用Mysql数据库，页面采用JSP技术。功能上支持医院管理、医生管理、病人管理、医院新闻管理。对这些模块的数据进行增删改查，是一个典型的医院管理系统。适合毕业设计、课程设计、大作业等！
